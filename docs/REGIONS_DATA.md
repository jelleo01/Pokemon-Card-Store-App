# 전국 시/군/구 데이터

## 📦 데이터 출처

### 추천: 행정안전부 공식 데이터
**행정구역(읍·면·동) 코드** — [공공데이터포털](https://www.data.go.kr/data/15077871/standard.do)
- 무료, CSV/JSON 다운로드
- 행정구역 변경 시 업데이트 (월 1회 정도)
- 형식:
  ```
  행정기관코드, 시도, 시군구, 읍면동
  1111000000,    서울특별시, 종로구, ""
  1111051500,    서울특별시, 종로구, 청운효자동
  ```

### 대안: 통계청 / 카카오 reverse geocoder
- 통계청 KOSIS — 더 자세한 통계용
- 카카오 reverse geocoder — 좌표 → 행정구역 (실시간), 정적 리스트는 제공 X

## 🔧 헬퍼 함수

```ts
// 시 이름으로 region 찾기
export function findRegion(city: string): Region | undefined {
  return REGIONS.find(r => r.city === city || r.cityFull === city);
}

// 카카오 reverse geocoder 결과 → REGIONS 형식 매핑
export function normalizeKakaoRegion(kakao: { region_1depth_name: string; region_2depth_name: string; }) {
  // 카카오는 "서울특별시" 반환 → REGIONS는 "서울" 사용
  const match = REGIONS.find(r => r.cityFull === kakao.region_1depth_name);
  if (!match) return null;
  return {
    city: match.city,
    district: kakao.region_2depth_name,
  };
}

// 모든 시 리스트
export const ALL_CITIES = REGIONS.map(r => r.city);
```

## 📌 MVP 베타 — 서울만 사용

서울 25개 구만 노출하려면:

```ts
export const BETA_REGIONS: Region[] = REGIONS.filter(r => r.city === '서울');
```

- 위치 검색 화면: 서울만 chip으로 표시
- 다른 지역 선택 시도하면: "서울 외 지역은 곧 오픈해요!" 토스트
